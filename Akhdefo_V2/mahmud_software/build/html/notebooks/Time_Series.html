Time Series¶

This program uses candiate velocity points from stackprep function and performs linear interpolation in time-domain to calibrate stacked velocity.

Additionally produces corrected timeseries velocity(daily) in a shapefile.

[2]:

from akhdefo_functions import Time_Series

Time_Series(stacked_raster_EW='raster_stackxn.tif', stacked_raster_NS='raster_stackyn.tif',
              velocity_points='stack_data/20220812_to_20221016_velocity.shp', dates_name='data/Names.txt', output_folder='stack_data/TS', rasteriz_mean_products=False,
              std=1 ,VEL_Scale='year', velocity_mode='linear', master_reference = True)
